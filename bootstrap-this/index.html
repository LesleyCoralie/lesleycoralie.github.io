Bootstrap Cards

A card is a flexible content container that can include contextual information such as a heading, a thumbnail image, and icons or buttons. Cards can be created in Bootstrap using very little markup or styling, but can provide all kinds of customization options. Cards are built with the Flexbox layout model, which allows for easy alignment.

Cards have no fixed width initially, and will fill the entire width of its parent container.

Carousel

A carousel is a slideshow component that allows users to cycle through a series of content. Built using a combination of CSS 3D transforms and JavaScript, the carousel with a series of images or text and comes with indicators and previous/next controls for ease of cycling.

To ensure that the carousel is visible, the .active class needs to be attributed to one of the slides. In addition, an id '#' needs to be set within the .carousel class in order additional slide controls for buttons and indicators.

Bootstrap Grid Layouts

Using a combination of flexbox and CSS grid modules, Bootstrap's Grid layout is built on a twelve column system with 5 break points, making it the ideal choice for creating mobile-first and responsive websites.

The Bootstrap Grid layout has three main components - containers, rows, and columns. Containers can be of a fixed width that snaps at certain breakpoints, or fluid - taking up 100% of the viewport width. Rows prepare web content for a layout, while columns store the content and are laid out with Bootstrap's 12-column system.

To prepare the Grid layout, add a a div with a class of .container followed by another div with a class of .row. Finally, create and nest the div .col class to create your columns. By default, the columns will be displayed with equal widths. However, columns can be customized using the .col-{number} class to set how many columns you would like an element to stretch.

Offsets

Want a bit more space for your web content and don't like the Grid format? Use offsets! An offset is used to push columns over to create more space between elements. You can offset grid columns in 2 ways - using the .offset-* (or .offset-{number}) grid class or margin utilities. In this example, we'll show you how to use the .offset- class.

You can move columns to the right by employing the .offset-* class. This class will increase the left margin of a column by whatever number of columns you assign. Examaple: an .offset-2 will move a .col two columns to the right.
